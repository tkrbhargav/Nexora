# Nexora

> A scalable React Admin & SaaS Boilerplate built with React, TypeScript, Tailwind CSS, shadcn/ui, TanStack Query, and TanStack Table.

Nexora is a modern frontend boilerplate designed to help developers build admin dashboards, SaaS platforms, internal tools, CRM systems, and business applications faster. It provides a clean architecture, reusable components, and a production-ready foundation without locking you into a specific backend.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- pnpm (recommended)

### Installation

```bash
git clone https://github.com/your-username/nexora.git

cd nexora

pnpm install
```

### Start Development Server

```bash
pnpm dev
```

### Build for Production

```bash
pnpm build
```

### Preview Production Build

```bash
pnpm preview
```
